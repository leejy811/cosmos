# Details

직접 프로젝트에 기여한 부분에 대한 상세 설명입니다.

전체 프로젝트 설명은 [문서 하단부](#manual)에서 확인 가능합니다.

------------------

### UI 애니메이션

코루틴과 트위닝 기법을 사용하여 자연스러우면서도 효율적으로 애니메이션을 구현하였습니다.

<img src="https://github.com/qkralsrl1029/Cosmos/blob/main/Demo/title.gif"  width="364" height="760"/>

-------------------

### EMP 기능

적들 이동 방향을 반전시킴으로써 넉백 효과를 구현하였습니다.

<img src="https://github.com/qkralsrl1029/Cosmos/blob/main/Demo/emp.gif"  width="364" height="364"/>

-------------------

### 이펙트

직접 구현한 애니메이션과 검증된 오픈소스 라이브러리 등을 활용하여 게임의 완성도를 더했습니다.

<img src="https://github.com/qkralsrl1029/Cosmos/blob/main/Demo/effect.gif"  width="364" height="760"/>

-------------------

### Bloom 이펙트

포스트 프로세싱 기능을 활용하여 게임 오버 이펙트를 연출하였습니다.

<img src="https://github.com/qkralsrl1029/Cosmos/blob/main/Demo/ending.gif"  width="364" height="760"/>
