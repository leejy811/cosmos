# Guideline

처음 게임 진입시 게임 팁이 표시됩니다. 이는 설정 창에서 다시 확인할 수 있습니다.


로비는 총 세 개의 화면으로 구성되어 있습니다.

가장 기본으로 타이틀 화면에서는, 게임 아이콘과 함께 플레이 기록을 확인할 수 있는 패널과 인게임으로 진입할 수 있는 버튼으로 구성되어 있습니다.
